# 📚 Estrutura de Dados - Materiais de Estudo

Este repositório foi criado com o intuito de organizar e documentar meus estudos práticos de **Estrutura de Dados**. Aqui, foco em implementações utilizando a linguagem **Python**.

---

## 🎯 Objetivo
Registrar a evolução do aprendizado em algoritmos e estruturas de armazenamento, servindo como base para consultas futuras e compartilhamento de conhecimento com colegas de curso.

## 📂 Conteúdo Atual
Até o momento, o repositório conta com:
* **Fila Estática (Linear e Circular):** Implementações focadas em gerenciamento de memória em arrays fixos.
* **Fila Dinâmica:** Estrutura baseada em nós encadeados que cresce conforme a necessidade.
* **Fila de Prioridade (Múltiplas Filas):** Sistema hierárquico que gerencia múltiplos fluxos simultaneamente (ex: triagem hospitalar).

## 🧪 Como Testar
Algumas estruturas possuem um arquivo de teste dedicado (como o `programa_teste.py`). Basta executá-lo - caso o arquivo de teste exista para essa estrutura - para validar as funções de inserção, remoção e visualização no console.
